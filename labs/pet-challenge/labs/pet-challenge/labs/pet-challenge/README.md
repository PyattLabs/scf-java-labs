# Pet Challenge

**COP2250 Java Programming | Week 1**

## Your Task

Build two Java classes:
- `Pet.java` — class with `name`, `age` fields and `speak()` method
- `PetShop.java` — creates Pet objects and calls speak()

## Expected Output
```
Buddy says: Woof! I am 3 years old.
Luna says: Woof! I am 5 years old.
```

## Commands
```bash
javac Pet.java PetShop.java
java PetShop
```
