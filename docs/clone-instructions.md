# Cloning the Class Repository

**COP2250 Java Programming**

All lab materials are in a Git repository. Clone it to your machine.

## Step 1: Open Terminal

- Mac: Terminal (Applications → Utilities)
- Windows: Git Bash (NOT PowerShell, NOT CMD)

## Step 2: Check Git is installed
```
git --version
```

## Step 3: Go to Desktop
```
cd ~/Desktop
```

## Step 4: Clone the repo
```
git clone https://github.com/PyattLabs/scf-java-labs.git
```

## Step 5: Go into the folder
```
cd scf-java-labs
```

## Step 6: See what's inside
```
ls
```

## Step 7: Explore labs
```
cd labs
ls
```

## To get updates later
```
cd ~/Desktop/scf-java-labs
git pull
```

## Windows users

If you don't have Git Bash, install Git from https://git-scm.com — Git Bash comes with it.
