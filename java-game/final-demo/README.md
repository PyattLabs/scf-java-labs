## Demo Requirements

Each team:
1. Screen-share terminal
2. `javac *.java` — must compile clean, no errors
3. `java Main` — game runs
4. Play through one complete game session
5. Demonstrate the end condition

---

## Individual Defense

Every team member is questioned individually.
You cannot point to your teammates. You answer for your own code.

**You will be asked:**
- Explain a class you wrote
- Why did you use an abstract class?
- Walk me through your game loop
- Where is exception handling in your game?
- Where does your interface appear?
- What would change if you added a GUI?

---

## GUI Reflection

Submit one paragraph per person to Canvas before class ends.

> *If you were to add a graphical user interface to this game, what Java framework would you use, how would the architecture need to change, and what would be the first three UI components you would build?*
