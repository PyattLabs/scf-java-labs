# Practice Quizzes

Self-study tools to prepare for weekly picker quizzes.

## How to Use

```bash
# Download the file for your current week
wget [raw link]

# Compile
javac PracticeQuizWeek5.java

# Run
java PracticeQuizWeek5
```

## Controls

| Key | Action |
|-----|--------|
| ENTER | Reveal answer / Next question |
| Y | Mark correct |
| N | Mark incorrect |
| R | Reshuffle and restart |
| Q | Quit |

## Available Quizzes

| File | Covers |
|------|--------|
| PracticeQuizWeek5.java | Terminal, Ch 1-4 |

New quizzes added each week. Content is cumulative.

## What's on the Picker Quiz?

- **Terminal commands:** pwd, ls, cd, javac, java, mkdir, touch
- **Chapter concepts:** T/F questions, short answer
- **DEMO questions:** Write code from memory

Practice until you can answer without hesitation.
